import { CopyButton } from "@godui/components";

The Copy Button writes a value to the clipboard, cross-fades its icon to a drawn checkmark, morphs the label from "Copy" to "Copied", and reverts after a timeout — with a non-secure-context fallback built in.

## Examples

### Icon only

Omit `children` for a compact, icon-only button. It falls back to an accessible `Copy` label.

## Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `value` | `string` | — | Text written to the clipboard |
| `variant` | `"default" \| "secondary" \| "outline" \| "ghost"` | `"outline"` | Visual style |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | Padding and text size |
| `timeout` | `number` | `1800` | ms before reverting to idle |
| `copiedLabel` | `ReactNode` | `"Copied"` | Label in the copied state |
| `onCopy` | `(value: string) => void` | — | Fires after a successful copy |
