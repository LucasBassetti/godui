A full-bleed atmospheric layer — radial glows, aurora dreams and soft pastel
washes. Drop it as the first child of a `relative overflow-hidden` container and
your content sits above it. Pick a look with the `variant` prop — 66 patterns
ported from [PatternCraft](https://patterncraft.fun).

## Usage

```tsx
import { EffectBackground } from "@/components/godui/effect-background";
```

The component renders an `absolute inset-0` layer at `z-base`. Place it inside a
positioned, clipped container and keep your content at `z-raised` or higher:

```tsx
<div className="relative overflow-hidden">
  <EffectBackground variant="aurora-dream-soft-harmony" />
  <YourContent />
</div>
```

Every pattern id is available from the `effectBackgroundVariants` array:

```tsx
import { effectBackgroundVariants } from "@godui/components";
```

## Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `variant` | `EffectBackgroundVariant` | `"aurora-dream-corner-whispers"` | One of 66 effect pattern ids (e.g. `"aurora-dream-soft-harmony"`, `"purple-radial-glow"`, `"cotton-candy-sky"`). See the [Storybook gallery](https://godui.design) or `effectBackgroundVariants` for the full set. |

All other `div` attributes (`className`, `style`, …) are forwarded to the layer.
