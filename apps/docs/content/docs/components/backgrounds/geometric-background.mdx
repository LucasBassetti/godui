A full-bleed geometric layer — grids, dashed grids, diagonal crosses and
mask-faded variants. Drop it as the first child of a `relative overflow-hidden`
container and your content sits above it. Pick a look with the `variant` prop —
99 patterns ported from [PatternCraft](https://patterncraft.fun).

## Usage

```tsx
import { GeometricBackground } from "@/components/godui/geometric-background";
```

The component renders an `absolute inset-0` layer at `z-base`. Place it inside a
positioned, clipped container and keep your content at `z-raised` or higher:

```tsx
<div className="relative overflow-hidden">
  <GeometricBackground variant="diagonal-cross-grid" />
  <YourContent />
</div>
```

Masked variants (e.g. `left-masked-basic-grid`) ship both `maskImage` and
`WebkitMaskImage`, so the fade works in Chromium and Safari. Every pattern id is
available from the `geometricBackgroundVariants` array:

```tsx
import { geometricBackgroundVariants } from "@godui/components";
```

## Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `variant` | `GeometricBackgroundVariant` | `"purple-gradient-grid-right"` | One of 99 geometric pattern ids (e.g. `"diagonal-cross-grid"`, `"dashed-grid-light"`, `"left-masked-basic-grid"`). See the [Storybook gallery](https://godui.design) or `geometricBackgroundVariants` for the full set. |

All other `div` attributes (`className`, `style`, …) are forwarded to the layer.
