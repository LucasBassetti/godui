A full-bleed gradient layer. Drop it as the first child of a `relative
overflow-hidden` container and your content sits above it. Pick a look with the
`variant` prop — 48 gradients ported from
[PatternCraft](https://patterncraft.fun).

## Usage

```tsx
import { GradientBackground } from "@/components/godui/gradient-background";
```

The component renders an `absolute inset-0` layer at `z-base`. Place it inside a
positioned, clipped container and keep your content at `z-raised` or higher:

```tsx
<div className="relative overflow-hidden">
  <GradientBackground variant="aurora-waves" />
  <YourContent />
</div>
```

Every pattern id is available from the `gradientBackgroundVariants` array if you
want to build a picker:

```tsx
import { gradientBackgroundVariants } from "@godui/components";
```

## Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `variant` | `GradientBackgroundVariant` | `"dark-radial-glow"` | One of 48 gradient pattern ids (e.g. `"aurora-waves"`, `"blue-radial-glow"`, `"crimson-shadow-glow"`). See the [Storybook gallery](https://godui.design) or `gradientBackgroundVariants` for the full set. |

All other `div` attributes (`className`, `style`, …) are forwarded to the layer.
