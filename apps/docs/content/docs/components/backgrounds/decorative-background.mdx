A full-bleed decorative wash — soft radial and corner gradients. Drop it as the
first child of a `relative overflow-hidden` container and your content sits
above it. Pick a look with the `variant` prop — 44 patterns ported from
[PatternCraft](https://patterncraft.fun).

## Usage

```tsx
import { DecorativeBackground } from "@/components/godui/decorative-background";
```

The component renders an `absolute inset-0` layer at `z-base`. Place it inside a
positioned, clipped container and keep your content at `z-raised` or higher:

```tsx
<div className="relative overflow-hidden">
  <DecorativeBackground variant="top-gradient-radial" />
  <YourContent />
</div>
```

Every pattern id is available from the `decorativeBackgroundVariants` array:

```tsx
import { decorativeBackgroundVariants } from "@godui/components";
```

## Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `variant` | `DecorativeBackgroundVariant` | `"top-gradient-radial"` | One of 44 decorative pattern ids (e.g. `"bottom-gradient-radial"`, `"purple-corner-grid"`). See the [Storybook gallery](https://godui.design) or `decorativeBackgroundVariants` for the full set. |

All other `div` attributes (`className`, `style`, …) are forwarded to the layer.
